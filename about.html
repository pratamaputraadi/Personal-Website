Universitas Singaperbangsa Karawang

Pendidikan

Saya merupakan lulusan SMK Trimitra dengan jurusan Teknik Komputer dan Jaringan (TKJ). Selama menempuh pendidikan di SMK, saya memperoleh pemahaman mendalam mengenai infrastruktur jaringan komputer, konfigurasi perangkat jaringan seperti router dan switch, serta dasar-dasar keamanan sistem. Selain itu, saya juga mendapatkan pengalaman praktik langsung dalam merancang dan mengelola jaringan, yang menjadi dasar kuat bagi ketertarikan saya di bidang teknologi informasi.

Saat ini, saya melanjutkan studi sebagai mahasiswa Semester 4 Informatika di Fakultas Ilmu Komputer, Universitas Singaperbangsa Karawang. Dalam perkuliahan, saya semakin memperdalam pemahaman tentang pemrograman, basis data, keamanan jaringan, serta teknologi informasi secara lebih luas. Saya terus mengembangkan keterampilan dalam networking dan pengembangan web, dengan tujuan untuk menguasai berbagai aspek dalam dunia IT dan menerapkannya dalam proyek serta penelitian yang saya jalani.

Minat

Saya memiliki minat yang besar dalam bidang jaringan komputer, khususnya dalam memahami arsitektur, konfigurasi, serta keamanan jaringan. Saya tertarik untuk mengeksplorasi bagaimana infrastruktur jaringan dapat dioptimalkan agar lebih efisien dan aman. Selain itu, saya juga mempelajari berbagai teknologi terkait, seperti manajemen server, cloud computing, dan keamanan siber, untuk memperdalam pemahaman saya dalam dunia networking.

Selain jaringan komputer, saya juga memiliki ketertarikan dalam bidang UI/UX (User Interface dan User Experience). Saya senang mendalami bagaimana desain dan pengalaman pengguna dapat mempengaruhi interaksi seseorang dengan sebuah sistem atau aplikasi. Saya terus belajar tentang prinsip-prinsip desain antarmuka yang intuitif, penggunaan warna dan tipografi yang efektif, serta bagaimana menciptakan pengalaman yang nyaman bagi pengguna dalam aplikasi berbasis web maupun mobile.

Pengalaman

Membangun Website Portofolio, Mengembangkan website pribadi sebagai media untuk menampilkan portofolio serta berbagi ilmu tentang pemrograman web. Dalam proses ini, saya menggunakan teknologi seperti HTML, CSS, JavaScript, serta framework modern seperti React.js atau Next.js untuk memastikan website interaktif dan responsif.
